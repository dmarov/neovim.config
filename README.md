# NVIMRC

## Set of configuration files for neovim

# Install:

1. Clone repository
```
git clone https://github.com/dmarov/nvimrc
```

2. Execute install script for your platform
```
cd nvimrc
```

```
zsh install-freebsd.sh
```

or

```
zsh install-arch-linux.sh
```

or

```
install-windows.ps1
```
